# Narayana School Website

A simple static website for Narayana School built with HTML and CSS.

## Features

- Clean, responsive design with blue header and green navigation
- Navigation menu with Home, About, Courses, and Contact sections
- About Us section describing the school's mission
- Courses offered: Science, Mathematics, Computer Science, English
- Contact information with email and phone

## How to View

Open `.vscode/index.html` in your web browser or serve it locally using a web server.

## Technologies Used

- HTML5
- CSS3
